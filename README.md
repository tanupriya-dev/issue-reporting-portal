The Issue Reporting Portal is a web-based application developed to help organizations efficiently report, track, and manage
employee issues. The system provides a centralized platform where employees can submit complaints and monitor their progress,
while administrators can manage users, update issue statuses, and oversee the entire issue resolution process.

Features
Employee Features
- Secure Login Authentication
- Report New Issues
- View Submitted Issues
- Track Issue Status
- Search and Filter Issues
- Dashboard Overview
- Logout Functionality

Admin Features
- Manage Employee Accounts
- View All Reported Issues
- Update Issue Status
- Monitor Dashboard Statistics
- Track Issue Resolution Progress
- Role-Based Access Control (RBAC)

Technologies Used
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- XAMPP
- phpMyAdmin

Project Modules
Authentication Module
Handles secure login, session management, and authorization.
Employee Dashboard
Provides issue statistics, recent issues, and quick navigation options.
Report Issue Module
Allows employees to submit issues with details such as title, category, priority, and description.
My Issues Module
Displays all issues reported by the employee with filtering and tracking options.
Admin Dashboard
Enables administrators to monitor issues, manage employees, and update issue statuses.
Database Module
Stores employee information, issue details, categories, notifications, and issue history.
